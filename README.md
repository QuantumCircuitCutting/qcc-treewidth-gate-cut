# qcc-treewidth-gate-cut

Treewidth-aware gate cut optimization

## Installation

    pip install qcc-treewidth-gate-cut

Or install the full suite:

    pip install quantum-circuit-cutting

## Quick Start

See [quickstart.ipynb](https://github.com/QuantumCircuitCutting/qcc-tutorials/blob/main/quickstart.ipynb)

## Development

    git clone https://github.com/QuantumCircuitCutting/qcc-treewidth-gate-cut.git
    cd qcc-treewidth-gate-cut
    pip install -e ".[dev]"
    pytest
